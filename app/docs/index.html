The Buckets Guide

NOTE: This guide is still very much a work in progress. If you find errors or have suggestions for topics or any other feedback send me an email at bugs@budgetwithbuckets.com or report a bug from within the Buckets app.

This guide is intended to help you learn how to budget your money well with an emphasis on the features Buckets includes for that purpose.

A good first step would be to start with the Getting Started articles.

This guide is available within the Buckets app as well as online at docs.budgetwithbuckets.com. And, any articles you print out will be available to read on paper :)
